# Privacy

- Video data uploaded to this application is stored locally on the server for analysis purposes only. Files are not transmitted to third parties (except OpenAI for transcription/LLM calls when an API key is provided).
- Do not upload videos containing personally identifiable information (PII) for hackathon submissions or public demos.
- Audio is extracted and sent to the OpenAI Whisper API for transcription when `OPENAI_API_KEY` is set. Review [OpenAI's data usage policy](https://openai.com/policies/api-data-usage-policies) for details.
